# 梦幻之心星的个人图床

## ！注意事项！
- 建议使用**复制**+**删除**操作来代替**移动**操作
- 移动操作只是修改文件的索引，不会创建文件副本
- 手动删除的文件可以通过数据恢复软件来找回
- Git删除的文件用数据恢复软件也很难找回

## 路径替换

- 本地临时路径前缀  
`G:/Pictrue_Bed_Class/${filename}/`

- 本地仓库路径前缀  
`G:/Pictrue_Bed/`

- 云端路径前缀  
`https://cdn.jsdelivr.net/gh/Sky-seeker/Pictrue_Bed/img/`

## 网页图片尺寸大小

- 建议不超过800x800
